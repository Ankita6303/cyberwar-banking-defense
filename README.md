# 💻 CyberWar Banking Defense

## 📌 Project Description

CyberWar Banking Defense is a full-stack web application designed to simulate and protect banking systems from cyber attacks. It helps users understand security threats and defensive mechanisms in a banking environment.

---

## 🚀 Technologies Used

### Frontend:

* React.js
* HTML, CSS, JavaScript

### Backend:

* Node.js
* Express.js

### Database:

* PostgreSQL

---

## ⚙️ Features

* User authentication system
* Real-time communication using Socket.io
* Secure backend APIs
* Database integration
* Interactive UI

---

## 🛠️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/YOUR-USERNAME/cyberwar-banking-defense.git
```

### 2️⃣ Backend setup

```bash
cd backend
npm install
npm start
```

### 3️⃣ Frontend setup

```bash
cd frontend
npm install
npm start
```

## 🔐 Environment Variables

Create a `.env` file in backend and add:

```
DB_HOST=your_host
DB_USER=your_user
DB_PASSWORD=your_password
DB_NAME=your_database
DB_PORT=your_port
```
